# 第零节 基础操作

1.空格和回车 ESC 都可以结束命令

2.基本都可以手动改长度 ,部分可以改宽度

3.右边可以缩放 移动等命令

4.上面可以做各种图和**注释** 等命令

5.指令p为小抓手（移动） z为缩放 delete为删除

**6.先选择你要操作的图形，再点击各种操作指令（移动 复制 缩放。。。）**

**7.CAD软件中[从右往左](http://jxahz.com/tag/3167/)与[从左往右](http://jxahz.com/tag/3168/)的[框选](http://jxahz.com/tag/3169/)有什么区别呢**

框选是指按住鼠标的左键拖动形成一个矩形区域进行对象选择。为充分对框选功能进行使用，CAD从右往左以及从左往右的框选对应了窗口模式与交叉（窗交）模式。从左往右框选的是窗口模式，图形中所有的顶点和边界都在矩形选框区域内才会被选中；从右往左框选的是交叉模式，图形中有任意一顶点或一边界在矩形选框范围内都会被选中。

CAD中对于这两种模式的使用有较明显的提示：首先，窗口框选边界是实线，交叉框选边界是虚线；窗口选框是蓝色的，交叉选框是绿色的，如下图所示：

# 第一节 直线 （不能改宽度）

1.空格和回车 ESC 都可以结束命令

2.可以手动改长度 ,不可以改宽度

3.指令l为画直线  l后c为自动连接端点进行密闭

4.F8为水平 或垂直的正交线



# 第二节 圆形 （不能改宽度）

1.空格和回车 ESC 都可以结束命令

2.可以手动改长度 ,不可以改宽度

3.指令c为画圆 后可默认指定半径（命令选项里有各种圆形）c后d为指定直径



# 第三节 圆弧 （不能改宽度）

1.空格和回车 ESC 都可以结束命令

2.可以手动改长度 ,不可以改宽度

3.arc为画圆弧 后r为输入半径

 

# 第四节 多段线 （能改宽度）

1.空格和回车 ESC 都可以结束命令

2.可以手动改长度 ,可以改宽度

3.pl为画多段线 后a为画多段圆弧后d可以改圆弧方向

4.选择不是多段线的图 后j 即转化为多段线 



# 第五节 多边形 （能改宽度）

1.空格和回车 ESC 都可以结束命令

2.可以手动改长度 ,可以改宽度

3.先输入边数 在选择内接 外切 （矩形不用）

4.矩形可以改圆角 当半径设为0是即无圆角

5.pol为画 多边形  后e直接作正多边形



# 第六节 移动和复制

1.**先点对象** 再点命令

2.m为移动 co为复制  注意以后的**阵列**



# 第七节 拉伸

1.拉伸一定要从右下角往左上角拉，**不框选全部**，那个选取框要是虚线才是拉伸,框选的部分是你要拉伸的部分，未框选的就不动！！！ **不能点选 和从左上角拉**！！！

观看本篇需注意：拉伸的不是用矩形工具画出来的矩形

关于为什么是平移：从左往右框选是平移，从右往左框选是拉伸

2.不可以单根单根图的拉伸



# 第八节 旋转 （从起点开始逆时针)

1.F8正交旋转

2.ro旋转 可以设置角度 再按c会把原图也留下

  按r是参照移动 先选两个你要转的原图的点 再选第三个你要转到的图的上的点



# 第九节 镜像 缩放 与 对齐缩放 

1.mi为镜像 然后点镜像线上的两点

2.sc为缩放 选择基点 以这个点为中心缩放 后c与旋转的差不多

r的话要选择你要缩放的图的两点 然后会循着这条边缩放

3.al为对齐缩放 选择你（两组）原图和想要去的地方的对应点 即可直接挪过去



# 第十节 修剪和延申

1.tr为修剪 先选择你要的对象 然后点你不要的对象 就剪掉了 （也可以直接剪）（可以进行拖拽 多个一起剪掉）

2.ex为延申 先选择你保持的对象 然后点你要延申的对象 就延申到你保持的对象了



# 第十一节 圆角和倒角

1.圆角通过设置半径控制边界出的圆润角度 当半径0是则无圆角90度相连

2.点击或f为画圆角命令后 r设置半径 再点击目标

3.倒角通过设置距离控制边界出的角度 当距离0是则无倒角直接90度相连

3.点击或cha为画倒角命令后 d设置两个距离 即倒角 的两个直角边长度 



# 第十二节 曲线（在“绘图”栏里）（关闭F）

1.可以对曲线上的点进行拖拽拉伸

2.曲线可以点击倒三角换为第一种或第二种模式

3.在“圆角”栏里有“光顺曲线”可以自动圆润连接两个曲线



# 第十二节 阵列

1.先选择你要创建**矩形阵列**的 对象 然后点击矩形阵列命令 就可以创建

2.然后可以设置阵列的多个属性

3.点最靠外的两个小三角 可以分别调行列数，点击右上角的正方形可以一起调

点右小角的正方形可以移动，点靠里面的两个小三角可以改间距



4.先选择你要创建**路径阵列**的 对象然后点击路径阵列命令再选择你要的路径曲线就可以创建

5.点矩阵命令中的基点再选择生成出的路径矩阵图中的点即可挪到曲线上 然后可以设置阵列的多个属性



6.先选择你要创建**环形阵列**的 对象 然后点击环形阵列命令 就可以创建

7.然后可以设置阵列的多个属性



# 第十三节 参数化（通过后期改参数 修改图形）

1**.选择命令后 选择的第一个图形 作为基础点 不会动 第二个图形 会做出相应改变**

2.不同的参数命令 可以叠加一起使用

3.对称命令中 最后选择对称轴	



4.线性约束做不到的 对齐约束可以做到既满足线性 又满足对齐

5.角度约束 第一个点的线也是不动的

6.转换 可以把注释改为约束！

7.加固 即把图形变成不动图
